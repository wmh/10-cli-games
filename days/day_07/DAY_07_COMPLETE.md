## 📝 遊戲特色

### 核心機制
1. **太空船控制**
   - 完整的360度旋轉控制
   - 慣性物理系統（太空中的真實感）
   - 推進加速效果
   - 緊急超空間傳送（有風險）

2. **小行星系統**
   - 三種尺寸：大、中、小
   - 擊中後分裂成更小的碎片
   - 越小的行星速度越快
   - 隨機運動軌跡

3. **戰鬥系統**
   - 子彈射擊（有冷卻時間）
   - 精確的碰撞檢測
   - 連擊加成系統（最高5倍）
   - 爆炸特效

4. **遊戲進程**
   - 關卡制度（清除所有小行星進入下一關）
   - 每關難度遞增
   - 生命系統（3條生命）
   - 重生無敵時間

### 物理引擎特點
- **向量運算**: 完整的2D向量類別實現
- **慣性移動**: 太空船保持速度直到施加推力
- **邊界環繞**: 從螢幕一側出去從另一側進來
- **圓形碰撞**: 準確的碰撞檢測系統
- **速度限制**: 防止過快移動導致的問題

## 🎯 操作方式
```
← → : 旋轉太空船
  ↑ : 向前推進
SPACE : 發射子彈
  ↓ : 超空間傳送（隨機位置，有10%機率毀滅）
ESC : 暫停遊戲
  Q : 退出
```

## 🏆 遊戲系統

### 計分規則
- 小行星（小）：100分
- 小行星（中）：50分
- 小行星（大）：20分
- 連擊加成：快速擊毀多個目標可獲得最高5倍加分

### 生命與關卡
- 初始生命：3條
- 碰撞小行星：失去1條生命
- 關卡進展：清空所有小行星後進入下一關
- 關卡難度：每關增加1個大型小行星（最多10個）

### 特殊機制
- **重生保護**: 每次重生有3秒無敵時間
- **關卡開始**: 每關開始有2秒無敵時間
- **超空間**: 緊急逃生，但有10%機率直接毀滅
- **視覺回饋**: 無敵時太空船閃爍
